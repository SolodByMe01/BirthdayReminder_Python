### Installation

1. Get a free app password for you google account here https://support.google.com/mail/answer/185833?hl=en
2. Clone the repo
   ```sh
   git clone https://github.com/SolodByMe01/BirthdayReminder_Python
   ```

   ```python
   python -m pip install pandas numpy python-dotenv datetime
   ```

   ```js
    receiver="ENTER RECEIVER ssantillanaq@gmail.com"
   ```
